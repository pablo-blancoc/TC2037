PASOS PARA CORRER EL PROGRAMA:
1. Activa el ambiente virtual de python usando el comando:
	>> source venv/bin/activate
2. Asegurate de que el archivo que quieras leer con el DFA esté dentro de la carpeta 'Brucelords'
3. Corre el archivo del autómata con el comando:
	>> python automata.py
4. Asegurate de cerrar el ambiente virtual de python con el comando:
	>> deactivate


Suerte!!!
